## 시작하기

**입찰공고 관리 시스템**은 한국 정부기관의 입찰공고를 자동으로 수집하여 효율적으로 관리할 수 있도록 도와주는 웹 기반 시스템입니다.

### 시스템 접속
1. 웹 브라우저에서 시스템 주소로 접속합니다
2. 로그인 정보를 입력하여 로그인합니다
3. 메인 대시보드가 표시됩니다

### 시스템 요구사항
- **브라우저**: Chrome, Firefox, Safari, Edge 최신 버전
- **해상도**: 1280x720 이상 권장
- **인터넷 연결**: 안정적인 인터넷 환경 필요

### 주요 기능 개요
- ✅ **자동 공고 수집**: 정부기관 웹사이트에서 자동으로 입찰공고 수집
- ✅ **실시간 검색**: 키워드, 기관, 카테고리별 실시간 검색
- ✅ **분류 관리**: 공고를 카테고리별로 자동 분류
- ✅ **입찰 추적**: 관심 있는 입찰의 진행 상황 추적
- ✅ **통계 분석**: 입찰 동향 및 통계 정보 제공
- ✅ **알림 기능**: 중요한 공고에 대한 알림

## 메인 화면

### 대시보드 구성
메인 대시보드는 다음과 같은 정보를 제공합니다:

```
┌─────────────────────────────────────────────────────┐
│                    상단 네비게이션                   │
├─────────────────────────────────────────────────────┤
│  📊 오늘의 통계    │  🔍 빠른 검색    │  ⚙️ 설정   │
├─────────────────────────────────────────────────────┤
│                   최신 공고 목록                     │
│  • [공사점검] 서울시 도로 안전점검용역 (서울시)      │
│  • [성능평가] 경기도 교육청 시설 성능평가 (경기도)   │
│  • [기타] 부산항 부두 시설 점검 (부산시)             │
├─────────────────────────────────────────────────────┤
│        📈 주요 통계         │      🎯 내 입찰 현황     │
│  • 오늘 새 공고: 48건       │  • 진행 중: 3건          │
│  • 이번 주 총 공고: 324건   │  • 완료됨: 12건          │
│  • 관심 키워드 매칭: 15건   │  • 낙찰: 2건             │
└─────────────────────────────────────────────────────┘
```

### 상단 네비게이션 메뉴
- **홈**: 메인 대시보드
- **공고**: 전체 공고 목록 및 검색
- **입찰**: 내 입찰 관리 (진행중, 완료, 낙찰)
- **통계**: 입찰 동향 및 분석 데이터
- **설정**: 시스템 환경설정

## 공고 조회

### 전체 공고 목록
**접속 경로**: 상단 메뉴 > 공고

공고 목록 화면에서는 다음과 같은 정보를 확인할 수 있습니다:

| 컬럼 | 설명 | 예시 |
|------|------|------|
| 제목 | 입찰공고 제목 | "서울시 도로 안전점검용역" |
| 기관명 | 발주 기관 | "서울특별시" |
| 카테고리 | 자동 분류된 카테고리 | "공사점검" |
| 등록일 | 공고 등록 날짜 | "2024-09-04" |
| 마감일 | 입찰 마감 날짜 | "2024-09-20" |
| 상태 | 입찰 진행 상태 | "진행중", "마감", "완료" |

### 공고 상세 보기
1. 목록에서 원하는 공고를 클릭합니다
2. 공고 상세 정보가 팝업 또는 새 페이지로 표시됩니다
3. 다음 정보를 확인할 수 있습니다:
   - **기본 정보**: 제목, 기관명, 담당자, 연락처
   - **입찰 일정**: 공고일, 마감일, 개찰일, 계약일정
   - **입찰 조건**: 참가자격, 입찰방법, 낙찰방법
   - **첨부 파일**: 입찰서류, 도면, 시방서 등
   - **상세 내용**: 공고 본문 내용

### 첨부 파일 다운로드
1. 공고 상세 화면에서 "첨부 파일" 섹션을 확인합니다
2. 다운로드할 파일명을 클릭합니다
3. 브라우저 다운로드가 시작됩니다

> ⚠️ **주의**: 첨부 파일은 발주기관 서버에서 직접 다운로드되므로 해당 기관의 정책에 따라 접근이 제한될 수 있습니다.

## 입찰 관리

### 관심 입찰 등록
관심 있는 입찰공고를 등록하여 추적 관리할 수 있습니다.

1. 공고 목록 또는 상세 화면에서 ⭐ 버튼을 클릭합니다
2. 입찰 상태를 선택합니다:
   - **관심있음**: 검토가 필요한 공고
   - **준비중**: 입찰서류 준비 중
   - **제출완료**: 입찰서류 제출 완료
   - **결과대기**: 개찰 결과 대기 중

### 입찰 현황 관리
**접속 경로**: 상단 메뉴 > 입찰

#### 진행 중 입찰
현재 진행 중인 입찰 목록을 확인합니다.
- 마감일까지 남은 시간 표시
- 중요도에 따른 색상 구분
- 준비 진행률 표시

#### 완료된 입찰
마감된 입찰들의 결과를 확인합니다.
- 낙찰자 정보
- 낙찰가격 (공개된 경우)
- 나의 순위 (참여한 경우)

#### 내 입찰 통계
- 월별 참여 현황
- 낙찰률 통계
- 주요 참여 분야 분석

### 알림 설정
중요한 입찰에 대한 알림을 받을 수 있습니다.

1. 공고 상세 화면에서 🔔 알림 버튼을 클릭합니다
2. 알림 조건을 설정합니다:
   - **마감 알림**: 마감 N일 전 알림
   - **결과 알림**: 개찰 결과 발표 시 알림
   - **변경 알림**: 공고 내용 변경 시 알림

## 검색 및 필터링

### 기본 검색
**검색 위치**: 모든 화면 상단의 검색창

1. 검색창에 키워드를 입력합니다
2. Enter 키를 누르거나 🔍 버튼을 클릭합니다
3. 검색 결과가 목록으로 표시됩니다

**검색 가능한 필드**:
- 공고 제목
- 기관명
- 공고 내용
- 담당자 정보

### 고급 검색
**접속 경로**: 공고 목록 > 고급 검색 버튼

더 정확한 검색을 위한 상세 조건을 설정할 수 있습니다.

#### 검색 조건
- **키워드**: 포함할 단어 또는 제외할 단어
- **기간**: 공고일, 마감일 기준 기간 설정
- **기관**: 특정 발주 기관 선택
- **카테고리**: 공사점검, 성능평가, 기타
- **지역**: 서울, 경기, 인천 등 지역별 필터
- **예산 규모**: 최소/최대 예산 범위

#### 검색 예시
```
키워드: "도로 AND 점검" NOT "설계"
기간: 2024-09-01 ~ 2024-09-30
기관: 서울특별시, 경기도
카테고리: 공사점검
예산: 1억원 이상
```

### 필터 기능

#### 빠른 필터
목록 화면에서 즉시 적용할 수 있는 필터들:

- **⏰ 오늘 등록**: 오늘 새로 등록된 공고만 표시
- **🔥 마감 임박**: 3일 내 마감되는 공고만 표시
- **⭐ 관심 공고**: 즐겨찾기한 공고만 표시
- **🎯 키워드 매칭**: 설정한 관심 키워드에 매칭되는 공고만 표시

#### 정렬 옵션
- **최신순**: 등록일 기준 최신 공고 우선
- **마감순**: 마감일이 가까운 공고 우선  
- **기관순**: 기관명 알파벳 순서
- **관련도순**: 검색 키워드와의 연관성 순서

### 저장된 검색
자주 사용하는 검색 조건을 저장하여 재사용할 수 있습니다.

1. 고급 검색에서 조건을 설정합니다
2. "검색 조건 저장" 버튼을 클릭합니다
3. 저장할 이름을 입력합니다 (예: "서울시 도로 점검")
4. 이후 "저장된 검색" 메뉴에서 빠르게 실행할 수 있습니다

## 통계 및 분석

**접속 경로**: 상단 메뉴 > 통계

### 대시보드 통계
실시간으로 업데이트되는 주요 지표들:

```
📊 오늘의 현황
├─ 새 공고: 48건 (전일 대비 +12%)
├─ 진행 중: 324건
├─ 이번 주 마감: 89건
└─ 내 관심 공고: 15건

🎯 카테고리별 분포
├─ 공사점검: 45% (146건)
├─ 성능평가: 32% (104건)  
├─ 기타: 23% (74건)

🏢 주요 발주기관 (이번 주)
├─ 서울특별시: 23건
├─ 경기도: 18건
├─ 인천광역시: 12건
└─ 기타 지자체: 56건
```

### 트렌드 분석
**차트 기능**:
- **월별 공고 수 추이**: 최근 12개월간 공고 발생 패턴
- **요일별 등록 패턴**: 어느 요일에 공고가 많이 등록되는지 분석
- **기관별 활동도**: 기관별 공고 발행 빈도
- **카테고리별 변화**: 분야별 공고 증감 추이

### 맞춤 분석 리포트
**월간/주간 리포트 생성**:
1. 통계 페이지에서 "리포트 생성" 버튼을 클릭합니다
2. 분석 기간을 선택합니다 (최근 1주일, 1개월, 3개월)
3. 포함할 항목을 선택합니다:
   - 전체 현황 요약
   - 관심 분야 동향
   - 경쟁 업체 분석
   - 시장 전망
4. PDF 또는 Excel 파일로 다운로드합니다

### 비교 분석
**전년도 동기 대비**:
- 공고 수 변화율
- 주요 발주기관 변화
- 예산 규모 변화
- 입찰 경쟁률 변화

## 설정

**접속 경로**: 상단 메뉴 > 설정

### 개인 설정

#### 프로필 정보
- 이름, 연락처, 이메일 수정
- 비밀번호 변경
- 소속 회사/기관 정보

#### 알림 설정
체크박스로 간편하게 설정 가능:

- ✅ **새 공고 알림**: 관심 키워드에 매칭되는 공고 등록 시
- ✅ **마감 임박 알림**: 관심 공고 마감 3일 전, 1일 전
- ❌ **결과 발표 알림**: 참여한 입찰의 결과 발표 시
- ✅ **시스템 공지**: 중요한 시스템 업데이트 및 공지사항

#### 화면 설정
- **테마**: 밝은 테마 / 어두운 테마
- **언어**: 한국어 (향후 영어 지원 예정)
- **시간대**: 한국 표준시 (KST)
- **페이지 크기**: 목록에서 한 번에 보여줄 항목 수 (20, 50, 100개)

### 관심 키워드 설정
자동 알림 및 필터링을 위한 키워드를 관리합니다.

#### 키워드 추가
1. "키워드 관리" 섹션에서 "+ 추가" 버튼을 클릭합니다
2. 키워드를 입력합니다 (예: "도로", "교량", "터널")
3. 중요도를 선택합니다: 🔴 높음, 🟡 보통, 🟢 낮음
4. "저장" 버튼을 클릭합니다

#### 키워드 조합
고급 사용자를 위한 조합 키워드:
- **AND 조건**: "도로 AND 점검" (둘 다 포함)
- **OR 조건**: "교량 OR 터널" (둘 중 하나 포함)  
- **NOT 조건**: "건설 NOT 설계" (설계 제외)

#### 키워드 점수
각 키워드에 점수를 부여하여 우선순위를 설정할 수 있습니다:
- **10점**: 매우 관심 높음 (즉시 알림)
- **7점**: 관심 높음 (하루 1회 요약)
- **5점**: 보통 관심 (주간 요약)
- **3점**: 참고용 (월간 요약)

### 대시보드 커스터마이징
메인 화면에 표시할 위젯을 선택할 수 있습니다.

#### 사용 가능한 위젯
- ✅ **최신 공고 목록**: 최근 등록된 공고 5-10건
- ✅ **내 입찰 현황**: 진행 중인 입찰 요약
- ✅ **오늘의 통계**: 주요 수치 요약  
- ❌ **날씨 정보**: 현재 위치 기반 날씨
- ✅ **시장 동향**: 업계 뉴스 및 동향
- ✅ **캘린더**: 중요한 마감일 표시

#### 레이아웃 설정
위젯의 크기와 위치를 드래그 앤 드롭으로 조정할 수 있습니다.

## FAQ

### 자주 묻는 질문

#### Q1. 새로운 공고는 언제 업데이트되나요?
**A**: 시스템은 매일 오전 10시와 오후 10시에 자동으로 각 기관 웹사이트를 확인하여 새로운 공고를 수집합니다. 긴급한 공고의 경우 수동으로 즉시 업데이트를 요청할 수도 있습니다.

#### Q2. 첨부 파일을 다운로드할 수 없어요.
**A**: 첨부 파일은 발주기관의 서버에서 직접 다운로드되므로, 해당 기관의 접근 정책이나 네트워크 상태에 따라 다운로드가 제한될 수 있습니다. 문제가 지속되면 해당 기관에 직접 문의하거나 고객지원팀에 연락주세요.

#### Q3. 알림이 오지 않아요.
**A**: 다음 사항을 확인해 주세요:
- 설정 > 알림 설정에서 해당 알림이 활성화되어 있는지 확인
- 이메일 주소가 정확한지 확인
- 스팸 메일함 확인
- 브라우저 알림 허용 설정 확인

#### Q4. 검색 결과가 너무 많아요.
**A**: 고급 검색을 이용하여 더 구체적인 조건을 설정해보세요:
- 기간을 좁혀서 검색
- 특정 기관이나 카테고리로 제한
- 제외할 키워드 사용 (예: "도로 NOT 설계")

#### Q5. 관심 키워드를 어떻게 설정해야 효과적인가요?
**A**: 
- **구체적인 키워드**: "건설" 보다는 "도로포장", "교량점검" 등
- **조합 사용**: "안전 AND (점검 OR 진단)"
- **점수 차등**: 핵심 키워드는 10점, 참고용은 3-5점
- **정기 검토**: 월 1회 키워드 효과성 검토 및 조정

#### Q6. 시스템 속도가 느려요.
**A**: 
- 브라우저 캐시 삭제 후 재접속
- 다른 브라우저로 테스트
- 대용량 첨부파일 다운로드 중이 아닌지 확인
- 문제가 지속되면 고객지원팀에 연락

#### Q7. 모바일에서도 사용할 수 있나요?
**A**: 네, 반응형 웹 디자인으로 모바일 기기에서도 최적화되어 사용하실 수 있습니다. 모바일 앱은 현재 개발 중이며 곧 출시될 예정입니다.

#### Q8. 데이터는 얼마나 오래 보관되나요?
**A**: 
- 공고 데이터: 3년간 보관
- 개인 활동 로그: 1년간 보관  
- 통계 데이터: 5년간 보관
- 삭제된 데이터는 복구가 불가능하므로 중요 데이터는 정기적으로 백업 권장

### 오류 해결

#### 로그인 관련 문제
**증상**: "로그인 실패" 또는 "세션 만료" 메시지
**해결방법**:
1. 비밀번호를 정확히 입력했는지 확인
2. Caps Lock이 켜져있지 않은지 확인
3. 브라우저 쿠키 및 캐시 삭제
4. 다른 브라우저로 시도
5. 비밀번호 찾기 기능 이용

#### 화면 표시 문제
**증상**: 화면이 깨지거나 버튼이 보이지 않음
**해결방법**:
1. 페이지 새로고침 (F5 또는 Ctrl+R)
2. 브라우저 확대/축소 레벨을 100%로 설정
3. 다른 브라우저로 테스트
4. 브라우저 업데이트 확인

#### 검색 결과 문제
**증상**: 검색해도 결과가 나오지 않음
**해결방법**:
1. 검색어 철자 확인
2. 더 일반적인 키워드로 다시 검색
3. 고급 검색에서 필터 조건 완화
4. 전체 기간으로 검색 범위 확장
